# OLIQ — ฟอร์ม Validation

ฟอร์มกรอกออนไลน์ (มือถือ/คอม) สำหรับงาน validation ผลิตภัณฑ์ OLIQ ในเนิร์สซิ่งโฮม
เปิด → ติ๊ก/พิมพ์ได้ → กดปุ่ม "พิมพ์ / บันทึก PDF" (1 ฟอร์มต่อ 1 ครั้ง)

- `index.html` — **ฟอร์มติดตามผล (Follow-up)** : ใช้ตอนกลับไปวัดผลหลังศูนย์ลองใช้แล้ว (ใช้ต่อไหม/ผลก่อน-หลัง/ขยับเข้าใกล้ซื้อ)
- `visit.html` — **ฟอร์มลงพื้นที่ครั้งแรก (Discovery)** : ใช้ตอนไปครั้งแรก (workflow/ใครจ่าย/sizing/pain)
